# Conglomerate

A library to serialize Ruby objects into collection+json.

![conglomerate](http://i.imgur.com/QkKZ0ru.jpg)

## Installation

Add this line to your application's Gemfile:

    gem 'conglomerate'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install conglomerate
